ZINGSA — Django URL name reference (Phase 1)
==============================================
Every link in templates/ uses {% url 'name' %} — never a hardcoded path.
Add matching `path(..., name='...')` entries to your Django urls.py so
these templates render without a NoReverseMatch error.

Anchors (e.g. #overview, #drone-spraying) are section IDs on the target
page, not separate URL names — they mirror the anchor pattern already
used in the existing static site (about.html#overview, etc.).

Suggested app: a single `pages` app is enough for Phase 1.
